敲敲云零代码平台

本地快速部署(敲敲云)
-----------------------------------

解压安装包：v2.2.0.zip

```
- 1.本地安装jdk7+
- 2.执行下面的Mysql脚本(自动创建数据库：qiaoqiaoyun)
   qiaoqiaoyun.sql
- 3.修改数据库和redis等配置
   修改 config/application-prod.yml，详见 https://help.qiaoqiaoyun.com/open/simpleStart.html
- 4.双击 start.bat 启动后台
- 5.访问敲敲云
   http://localhost
   账号密码：admin/123456
```
